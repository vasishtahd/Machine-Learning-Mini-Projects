Execution instructions:

1) Firstly, Run the following command on console to ensure you have all the requirements.

pip install -r requirements.txt

2) Ensure that the 2015.csv, 2016.csv and 2017.csv are in the same directory 
   as the "team_assign1.py" file.

3) Ensure that Python version >=3.6.

4) Use any IDE ( We have used Spyder) to execute the "team_assign1.py" file.

5) Look into the console for all the relevant outputs.
